API for generated scripts

  • server=...&dataset=...&return=script-options
  • ?server=...&dataset=...&return=script&format=SCRIPT_OPTION where SCRIPT_OPTION is a value in the array returned by first request.

    • Examples

  • Request JSON object with list of script languages
  • Request script with no parameters, start, and stop. Script will request all parameters over default time range.
  • Request a script for a two parameters over a time range given by start and stop.
